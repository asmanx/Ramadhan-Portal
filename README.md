# Ramadhan-Portal
Website Ramadhan Portal

Website ini dilengkapi beberapa fitur diantaranya:
1. Doa di bulan Ramadhan
   Berisi niat puasa dan juga doa berbuka puasa serta terdapat penjelasan singkat tentang keutamaan bulan Ramadhan
   
3. Menghitung Zikir
   Fitur ini memungkin pengguna untuk melakukan zikir, dimana fitur ini dapat memasukkan target zikir yang ingin dicapai oleh     pengguna dan jika targetnya sudah tercapai maka akan ada pesan Apresiasi untuk pencapaian tersebut
   
5. Menghitung Zakat
   Fitur ini memiliki 2 jenis zakat, yakni zakat penghasilan dan juga zakat emas sehingga memungkin pengguna untuk menghitung     zakat yang akan di keluarkan sesuai jenis yang dipilih. 
   
7. Jadwal Imsak
   fitur ini memungkinkan pengguna untuk melihat jadwal imsak di kota yang di pilih oleh pengguna

9. To-Do-List
    Berisikan tracker ibadah diantaranya (tracker solat, baca al-quran dan juga puasa)
